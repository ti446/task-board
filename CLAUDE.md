# タスクボード プロジェクト

## プロジェクト概要
SAMURAIスプリントで制作するタスクボードのWebアプリです。

## 技術スタック
- HTML
- CSS
- JavaScript

## コーディング規約
- コメントは日本語で書く
- 変数名は英語のキャメルケースを使う（例：`userName`、`itemList`）
- インデントはスペース2文字を使う

## ファイル構成
```
task-board/
├── index.html
├── style.css
└── script.js
```

## Git運用ルール

### 基本方針
- コードを変更するたびに、変更内容をコミットしてGitHubにプッシュすること
- コミットは細かい単位で行い、1つの変更につき1コミットを原則とする

### コミットメッセージ
- 日本語で書く
- 変更内容が一目でわかる簡潔な文にする
- 例：`タスク追加機能を実装`、`削除ボタンのスタイルを修正`

### 手順
1. ファイルを変更する
2. `git add` で変更をステージングする
3. `git commit -m "変更内容"` でコミットする
4. `git push origin main` でGitHubにプッシュする

## 返答ルール
- 返答は必ず日本語で行う
